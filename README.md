# Temperature Converter

This project is developed as part of the Prodigy InfoTech Software Development Internship.

## Description
The Temperature Converter is a simple Python program that converts temperature values between Celsius, Fahrenheit, and Kelvin. The program takes user input for the temperature value and unit, then converts it into the other temperature scales.

## Features
- Converts Celsius to Fahrenheit and Kelvin
- Converts Fahrenheit to Celsius and Kelvin
- Converts Kelvin to Celsius and Fahrenheit
- User-friendly input and output
- Demonstrates use of conditional statements and formulas

## Technologies Used
- Python

## How to Run the Program
1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder.
4. Run the program using:
   python temperature_converter.py
